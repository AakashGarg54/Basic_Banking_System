About Spark Foundation Bank

This application is made by Aakash Garg . I am a web developer intern at Spark Foundation. According to the guidelines, I have built a Basic Banking Web app using Flask as a framework, SQLite3 as a database and Bootstrap as UI framework.

According to the given question, The flow of the web app is Home Page > View all Customers > Select and View one Customer > Transfer Money > Select Customer to transfer to > View all Customers. Apart from given question, I have add a new feature to my application i.e. Transaction History. User can see all the transaction of the Bank as well as there own personal transactions. Few user are already added in the application with some transaction in the database.
